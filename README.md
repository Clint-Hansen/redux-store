# Shop-Shop Store (Redux)

## Description

An online store where the user may purchase a variety of items.  Items may be added and removed from the cart and the user can checkout using a third-party pamyent processing API.

## Built With

* GraphQl
* Apollo
* MongoDB
* Heroku
* React
* Stripe
